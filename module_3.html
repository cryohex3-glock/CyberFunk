HTML Tables

Tables in HTML are used to display structured data in rows and columns. They are created using a set of related tags: <table>, <tr>, <th>, and <td>.

Figure 3: HTML tables and forms used in a web application

Table Tags Explained

  • <table> – Wraps the entire table
  • <thead> – Groups the header rows of the table
  • <tbody> – Groups the body content rows
  • <tr> – Defines a table row
  • <th> – Defines a header cell (bold and centered by default)
  • <td> – Defines a standard data cell
  • colspan / rowspan – Attributes to merge cells across columns or rows

HTML Forms

Forms allow users to input data that is submitted to a server for processing. A form contains form elements such as text fields, checkboxes, radio buttons, and submit buttons, all wrapped in a <form> tag.

Common Form Input Types

  1. text – Single-line text input
  2. email – Email address input with built-in validation
  3. password – Masked text input
  4. number – Numeric input
  5. tel – Telephone number input
  6. radio – Select one option from a group
  7. checkbox – Toggle one or more options
  8. select – Dropdown list
  9. textarea – Multi-line text input
  10. submit – Button to submit the form

Form Attributes Reference

Attribute Used On Purpose
action <form> URL where the form data is sent on submission
method <form> HTTP method: GET or POST
name <input> Identifies the field when data is submitted
placeholder <input> Hint text shown when the field is empty
required <input> Makes the field mandatory before submission
type <input> Specifies the kind of input (text, email, password, etc.)
