What is HTML?

"HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure of a web page sing a series of elements represented by tags. HTML is the skeleton of every website on the internet."

Figure 1: Basic Structure of an HTML Document

Core HTML Elements

HTML uses tags enclosed in angle brackets to define elements. Most elements have an opening and a closing tag. Here are the most important ones:

  • ::marker <html> " -Root element of an HTML page."
  • ::marker <head> " -Contains meta-information about the document, such as title and links to stylesheets."
  • ::marker <title> " -Defines the title of the document, shown in the browser's title bar or tab."
  • ::marker <body> " -Contains the content of the document, such as text, images, and links."
  • ::marker <h1> to <h6> " -Heading tags, where <h1> is the highest level and <h6> is the lowest."
  • ::marker <p> " -Defines a paragraph of text."
  • ::marker <a> " -Defines a hyperlink to another page or resource."
  • ::marker <img> " -Embeds an image in the document."
  • ::marker <ul> " -Defines an unordered list."
  • ::marker <ol> " -Defines an ordered list."
  • ::marker <li> " -Defines a list item."
  • ::marker <div> " -Defines a division or section in the document, often used for layout purposes."
  • ::marker <span> " -Defines a small section of text, often used for styling purposes."

HTML Document Structure

Element Description

<head> Contains meta-information about the document, such as title and links to stylesheets.
<title> Defines the title of the document, shown in the browser's title bar or tab.
