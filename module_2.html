What is CSS?

"CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS controls the layout, colors, fonts, and overall visual appearance of web pages. It allows developers to create visually appealing and responsive designs."

Figure 1: Example of CSS Styling

Learning Objectives

By the end of this module, you will be able to:

  1. Understand the role of CSS in web development.
  2. Apply basic CSS properties to style HTML elements.
  3. Create responsive layouts using CSS techniques.

Ways to Apply CSS

  • ::marker Inline CSS: Apply styles directly to HTML elements using the styleattribute.
  • ::marker Internal CSS: Define styles within a <style> tag in the <head> section of the HTML document.
  • ::marker External CSS: Link to an external CSS file using the <link> tag.

CSS Properties Reference

Property Description
color Sets the text color of an element.
background-color Sets the background color of an element.
font-size Sets the font size of text.
margin Sets the margin area on all sides of an element.
padding Sets the padding area on all sides of an element.
